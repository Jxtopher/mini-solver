# Mini Solver

Le meilleur, par Dr. Cricri

# Build

Simply run:

```
make
```

However if you use Mac OS X and some G++ flags don't exist, try:

```
make buildOSX
```

# Run

To run the default program:

```
make run
```

# Test

To run the tests:

```
make unitTest
```

However if you use Mac OS X and some G++ flags don't exist, try:

```
make unitTestOSX
```
