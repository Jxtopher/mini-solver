# Build

Simply run:

```
make
```

However if you use Mac OS X and some G++ flags don't exist, try:

```
make build-osx
```

# Run

To run the default program:

```
make run
```

# Test

To run the tests:

```
make unit-test
```

However if you use Mac OS X and some G++ flags don't exist, try:

```
make unit-test-osx
```
